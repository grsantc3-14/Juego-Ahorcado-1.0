Qué hace el programa? Juego del ahorcado, te pregunta tu nombre para empezar,
puedes elegir entre serie de palabras de colores o animales,
después tienes que tienes que colocar una letra a la vez para adivinar la palabra secreta,
te dice cuantas llevas vidas y cuantas rondas.

Qué partes del código implementaste tú? Lo elabore yo solo.

Qué aprendiste durante el desarrollo? Aprendí a mejorar mis funciones y que es fácil utilizar Tkinter, 
a realizar una interfaz del usuario, me ayudo a comprender conceptos abstractos de la programación básica.
